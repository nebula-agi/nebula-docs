# Index Management

The Indices API allows you to create and manage search indices in the R2R platform. Search indices optimize document retrieval by providing fast access to document content and metadata.

## Overview

The Indices API provides:

- **Index Creation**: Create new search indices
- **Index Management**: List, retrieve, and delete indices
- **Optimization**: Optimize search performance
- **Configuration**: Configure index settings and parameters

## Endpoints

### Index Management

- `POST /indices` - Create index
- `GET /indices` - List indices
- `GET /indices/{id}` - Get index
- `DELETE /indices/{id}` - Delete index

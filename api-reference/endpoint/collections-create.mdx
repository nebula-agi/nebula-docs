# Create Collection

Create a new collection to organize your documents.

<RequestExample>
```bash
curl -X POST "https://api.r2r.ai/v3/collections" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "name": "My New Collection",
    "description": "This is a sample collection"
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "name": "My New Collection",
    "description": "This is a sample collection",
    "owner_id": "789a012b-c34d-5e6f-g789-012345678901",
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T10:30:00Z"
  }
}
```
</ResponseExample>

**Description:** Creates a new collection with the specified name and optional description. The authenticated user becomes the owner of the collection.

**Authentication:** Required

**Request Body:**
- `name` (string, required): The name of the collection
- `description` (string, optional): An optional description of the collection

**Response Fields:**
- `id` (string): Unique identifier for the collection
- `name` (string): Collection name
- `description` (string): Collection description
- `owner_id` (string): ID of the collection owner
- `created_at` (string): ISO 8601 timestamp when the collection was created
- `updated_at` (string): ISO 8601 timestamp when the collection was last updated

## Error Codes

| Code | Description |
|------|-------------|
| `400` | Bad Request - Invalid request body |
| `401` | Unauthorized - Invalid or missing API key |
| `409` | Conflict - Collection with this name already exists |
| `500` | Internal Server Error - System error |

## Rate Limiting

Collection creation: Rate limited

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# Create a new collection
result = client.collections.create(
    name="My New Collection",
    description="This is a sample collection"
)
print(f"Created collection: {result['id']}")
```

### JavaScript

```javascript
const { r2rClient } = require("r2r-js");

const client = new r2rClient("your-api-key");

// Create a new collection
const response = await client.collections.create({
    name: "My New Collection",
    description: "This is a sample collection"
});
console.log(`Created collection: ${response.id}`);
```

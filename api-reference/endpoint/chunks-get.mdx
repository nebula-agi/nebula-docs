# Get Chunk

Retrieve details of a specific chunk by ID.

<RequestExample>
```bash
curl -X GET "https://api.r2r.ai/v3/chunks/b4ac4dd6-5f27-596e-a55b-7cf242ca30aa" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "id": "b4ac4dd6-5f27-596e-a55b-7cf242ca30aa",
    "text": "This is the chunk content...",
    "metadata": {
      "document_id": "doc-123",
      "chunk_index": 5,
      "page_number": 2
    },
    "embedding": [0.1, 0.2, 0.3, ...],
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T10:30:00Z"
  }
}
```
</ResponseExample>

**Description:** Returns detailed information about a specific chunk, including its text content, metadata, and vector embedding.

**Authentication:** Required

**Path Parameters:**
- `id` (string, required): The unique identifier of the chunk

**Response Fields:**
- `id` (string): Unique identifier for the chunk
- `text` (string): Chunk text content
- `metadata` (object): Chunk metadata
- `embedding` (array): Vector embedding
- `created_at` (string): ISO 8601 timestamp when the chunk was created
- `updated_at` (string): ISO 8601 timestamp when the chunk was last updated

## Error Codes

| Code | Description |
|------|-------------|
| `404` | Not Found - Chunk does not exist |
| `403` | Forbidden - User does not have access to this chunk |
| `401` | Unauthorized - Invalid or missing API key |
| `500` | Internal Server Error - System error |

## Rate Limiting

Chunk retrieval: Rate limited

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# Get chunk details
result = client.chunks.retrieve("b4ac4dd6-5f27-596e-a55b-7cf242ca30aa")
print(f"Chunk text: {result['text']}")
```

### JavaScript

```javascript
const { r2rClient } = require("r2r-js");

const client = new r2rClient("your-api-key");

// Get chunk details
const response = await client.chunks.retrieve({
    id: "b4ac4dd6-5f27-596e-a55b-7cf242ca30aa"
});
console.log(`Chunk text: ${response.text}`);
```

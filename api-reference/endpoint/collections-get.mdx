# Get Collection

Retrieve details of a specific collection by ID.

<RequestExample>
```bash
curl -X GET "https://api.r2r.ai/v3/collections/123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "name": "My Collection",
    "description": "A sample collection",
    "owner_id": "789a012b-c34d-5e6f-g789-012345678901",
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T10:30:00Z"
  }
}
```
</ResponseExample>

**Description:** Returns detailed information about a specific collection. Users can access collections they own or collections they are members of.

**Authentication:** Required

**Path Parameters:**
- `id` (string, required): The unique identifier of the collection

**Response Fields:**
- `id` (string): Unique identifier for the collection
- `name` (string): Collection name
- `description` (string): Collection description
- `owner_id` (string): ID of the collection owner
- `created_at` (string): ISO 8601 timestamp when the collection was created
- `updated_at` (string): ISO 8601 timestamp when the collection was last updated

## Error Codes

| Code | Description |
|------|-------------|
| `404` | Not Found - Collection does not exist |
| `403` | Forbidden - User does not have access to this collection |
| `401` | Unauthorized - Invalid or missing API key |
| `500` | Internal Server Error - System error |

## Rate Limiting

Collection retrieval: Rate limited

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# Get collection details
result = client.collections.retrieve("123e4567-e89b-12d3-a456-426614174000")
print(f"Collection name: {result['name']}")
```

### JavaScript

```javascript
const { r2rClient } = require("r2r-js");

const client = new r2rClient("your-api-key");

// Get collection details
const response = await client.collections.retrieve({
    id: "123e4567-e89b-12d3-a456-426614174000"
});
console.log(`Collection name: ${response.name}`);
```

# Get Configuration

Retrieve the current system configuration.

<RequestExample>
```bash
curl -X GET "https://api.r2r.ai/v3/config" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "app_name": "R2R API",
    "version": "1.0.0",
    "environment": "production",
    "features": {
      "auth_enabled": true,
      "rate_limiting": true,
      "logging": true
    },
    "limits": {
      "max_file_size": "100MB",
      "max_concurrent_requests": 100
    }
  }
}
```
</ResponseExample>

**Description:** Returns the current system configuration including feature flags, limits, and environment settings.

**Authentication:** Required

**Response Fields:**
- `app_name` (string): Application name
- `version` (string): System version
- `environment` (string): Environment (production, staging, development)
- `features` (object): Feature flags and capabilities
- `limits` (object): System limits and constraints

## Error Codes

| Code | Description |
|------|-------------|
| `401` | Unauthorized - Invalid or missing API key |
| `403` | Forbidden - Insufficient permissions |
| `500` | Internal Server Error - System error |

## Rate Limiting

Configuration and status: 10 requests per minute

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# Get configuration
config = client.system.config()
print(f"Version: {config['version']}")
```

### JavaScript

```javascript
const { R2RClient } = require('r2r-js');

const client = new R2RClient('your-api-key');

// Get configuration
const config = await client.system.config();
console.log(`Version: ${config.version}`);
```

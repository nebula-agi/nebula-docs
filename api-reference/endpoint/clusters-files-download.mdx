# POST `/clusters/files/download`

Downloads a file from a cluster by file ID or name.

## Request
- **Body (JSON):**
  - `cluster` (string, required): Cluster name or ID
  - `fileId` (string, optional): File ID
  - `fileName` (string, optional): File name
- **Headers:**
  - `Authorization: Bearer <token>` or `Authorization: ApiKey <key>`

### Example (cURL)
```bash
curl -X POST https://api.nebulacloud.app/clusters/files/download \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"cluster": "my-cluster", "fileName": "document.pdf"}'
```

## Response
- **Content:** The file is returned as a binary stream with appropriate `Content-Disposition` and `Content-Type` headers.

## Error Responses
- `404`: File or cluster not found
- `500`: Internal server error

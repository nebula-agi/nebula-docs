# Update Collection

Update the name and description of a collection.

<RequestExample>
```bash
curl -X POST "https://api.r2r.ai/v3/collections/123e4567-e89b-12d3-a456-426614174000" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "name": "Updated Collection Name",
    "description": "Updated description"
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "name": "Updated Collection Name",
    "description": "Updated description",
    "owner_id": "789a012b-c34d-5e6f-g789-012345678901",
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T11:30:00Z"
  }
}
```
</ResponseExample>

**Description:** Updates the name and/or description of a collection. Only the collection owner can update the collection.

**Authentication:** Required

**Path Parameters:**
- `id` (string, required): The unique identifier of the collection to update

**Request Body:**
- `name` (string, optional): The new name of the collection
- `description` (string, optional): The new description of the collection
- `generate_description` (boolean, optional, default: false): Whether to generate a new synthetic description for the collection

**Response Fields:**
- `id` (string): Unique identifier for the collection
- `name` (string): Updated collection name
- `description` (string): Updated collection description
- `owner_id` (string): ID of the collection owner
- `created_at` (string): ISO 8601 timestamp when the collection was created
- `updated_at` (string): ISO 8601 timestamp when the collection was last updated

## Error Codes

| Code | Description |
|------|-------------|
| `400` | Bad Request - Invalid request body |
| `404` | Not Found - Collection does not exist |
| `403` | Forbidden - User is not the owner of this collection |
| `401` | Unauthorized - Invalid or missing API key |
| `500` | Internal Server Error - System error |

## Rate Limiting

Collection updates: Rate limited

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# Update collection
result = client.collections.update(
    "123e4567-e89b-12d3-a456-426614174000",
    name="Updated Collection Name",
    description="Updated description"
)
print(f"Updated collection: {result['name']}")
```

### JavaScript

```javascript
const { r2rClient } = require("r2r-js");

const client = new r2rClient("your-api-key");

// Update collection
const response = await client.collections.update({
    id: "123e4567-e89b-12d3-a456-426614174000",
    name: "Updated Collection Name",
    description: "Updated description"
});
console.log(`Updated collection: ${response.name}`);
```

# List Chunks

Retrieve a list of chunks with optional filtering.

<RequestExample>
```bash
curl -X GET "https://api.r2r.ai/v3/chunks?offset=0&limit=10&include_vectors=false" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "results": [
      {
        "id": "chunk-123",
        "text": "This is the chunk content...",
        "metadata": {
          "document_id": "doc-123",
          "chunk_index": 5
        },
        "embedding": [0.1, 0.2, 0.3, ...],
        "created_at": "2024-01-15T10:30:00Z",
        "updated_at": "2024-01-15T10:30:00Z"
      }
    ],
    "total": 1,
    "offset": 0,
    "limit": 10
  }
}
```
</ResponseExample>

**Description:** Returns a paginated list of chunks that the authenticated user has access to. Users can see chunks from documents they own or documents in collections they are members of.

**Authentication:** Required

**Query Parameters:**
- `metadata_filter` (string, optional): Filter by metadata in JSON format
- `include_vectors` (boolean, optional, default: false): Whether to include vector embeddings in the response
- `offset` (integer, optional, default: 0): Number of objects to skip. Must be >= 0.
- `limit` (integer, optional, default: 100): Limit on the number of objects to return. Must be between 1 and 1000.

**Response Fields:**
- `results` (array): Array of chunk objects
  - `id` (string): Unique identifier for the chunk
  - `text` (string): Chunk text content
  - `metadata` (object): Chunk metadata
  - `embedding` (array): Vector embedding (if include_vectors=true)
  - `created_at` (string): ISO 8601 timestamp when the chunk was created
  - `updated_at` (string): ISO 8601 timestamp when the chunk was last updated
- `total` (integer): Total number of chunks
- `offset` (integer): Current offset
- `limit` (integer): Current limit

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# List chunks
result = client.chunks.list(
    metadata_filter={"document_id": "doc-123"},
    include_vectors=False,
    offset=0,
    limit=10
)
print(f"Found {result['total']} chunks")
```

### JavaScript

```javascript
const { r2rClient } = require("r2r-js");

const client = new r2rClient("your-api-key");

// List chunks
const response = await client.chunks.list({
    metadataFilter: { document_id: "doc-123" },
    includeVectors: false,
    offset: 0,
    limit: 10
});
console.log(`Found ${response.total} chunks`);
```

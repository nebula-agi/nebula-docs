# Get System Status

Get detailed system status and metrics.

<RequestExample>
```bash
curl -X GET "https://api.r2r.ai/v3/status" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "system": {
      "uptime": "24h 30m 15s",
      "memory_usage": "75%",
      "cpu_usage": "45%",
      "disk_usage": "60%"
    },
    "services": {
      "database": "healthy",
      "vector_store": "healthy",
      "embedding_service": "healthy",
      "llm_service": "healthy"
    },
    "metrics": {
      "total_requests": 15420,
      "active_users": 125,
      "documents_processed": 8920
    }
  }
}
```
</ResponseExample>

**Endpoint:** `GET /status`

**Description:** Returns detailed system status including uptime, resource usage, service health, and performance metrics.

**Authentication:** Required

**Response Fields:**
- `system` (object): System-level metrics
  - `uptime` (string): System uptime
  - `memory_usage` (string): Memory usage percentage
  - `cpu_usage` (string): CPU usage percentage
  - `disk_usage` (string): Disk usage percentage
- `services` (object): Health status of individual services
- `metrics` (object): Performance and usage metrics

## Error Codes

| Code | Description |
|------|-------------|
| `401` | Unauthorized - Invalid or missing API key |
| `403` | Forbidden - Insufficient permissions |
| `500` | Internal Server Error - System error |

## Rate Limiting

Configuration and status: 10 requests per minute

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# Get system status
status = client.system.status()
print(f"Uptime: {status['system']['uptime']}")
```

### JavaScript

```javascript
const { R2RClient } = require('r2r-js');

const client = new R2RClient('your-api-key');

// Get system status
const status = await client.system.status();
console.log(`Uptime: ${status.system.uptime}`);
```

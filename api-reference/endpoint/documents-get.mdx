# Get Document

Retrieve details of a specific document by ID.

<RequestExample>
```bash
curl -X GET "https://api.r2r.ai/v3/documents/b4ac4dd6-5f27-596e-a55b-7cf242ca30aa" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "id": "b4ac4dd6-5f27-596e-a55b-7cf242ca30aa",
    "title": "Sample Document",
    "content": "This is the document content...",
    "metadata": {
      "author": "John Doe",
      "created_date": "2024-01-15"
    },
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T10:30:00Z"
  }
}
```
</ResponseExample>

**Description:** Returns detailed information about a specific document. Users can access documents they own or documents in collections they are members of.

**Authentication:** Required

**Path Parameters:**
- `id` (string, required): The unique identifier of the document

**Response Fields:**
- `id` (string): Unique identifier for the document
- `title` (string): Document title
- `content` (string): Document content
- `metadata` (object): Document metadata
- `created_at` (string): ISO 8601 timestamp when the document was created
- `updated_at` (string): ISO 8601 timestamp when the document was last updated

## Error Codes

| Code | Description |
|------|-------------|
| `404` | Not Found - Document does not exist |
| `403` | Forbidden - User does not have access to this document |
| `401` | Unauthorized - Invalid or missing API key |
| `500` | Internal Server Error - System error |

## Rate Limiting

Document retrieval: Rate limited

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# Get document details
result = client.documents.retrieve("b4ac4dd6-5f27-596e-a55b-7cf242ca30aa")
print(f"Document title: {result['title']}")
```

### JavaScript

```javascript
const { r2rClient } = require("r2r-js");

const client = new r2rClient("your-api-key");

// Get document details
const response = await client.documents.retrieve({
    id: "b4ac4dd6-5f27-596e-a55b-7cf242ca30aa"
});
console.log(`Document title: ${response.title}`);
```

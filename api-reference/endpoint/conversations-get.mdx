# Get Conversation

Retrieve details of a specific conversation by ID.

<RequestExample>
```bash
curl -X GET "https://api.r2r.ai/v3/conversations/123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "name": "My Conversation",
    "user_id": "user-123",
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T10:30:00Z",
    "messages": [
      {
        "id": "msg-123",
        "content": "Hello, how can I help you?",
        "role": "assistant",
        "parent_id": null,
        "metadata": {},
        "created_at": "2024-01-15T10:30:00Z"
      }
    ]
  }
}
```
</ResponseExample>

**Description:** Returns detailed information about a specific conversation including all its messages. Users can only access conversations they own.

**Authentication:** Required

**Path Parameters:**
- `id` (string, required): The unique identifier of the conversation

**Response Fields:**
- `id` (string): Unique identifier for the conversation
- `name` (string): Conversation name
- `user_id` (string): ID of the user who owns the conversation
- `created_at` (string): ISO 8601 timestamp when the conversation was created
- `updated_at` (string): ISO 8601 timestamp when the conversation was last updated
- `messages` (array): Array of message objects
  - `id` (string): Unique identifier for the message
  - `content` (string): Message content
  - `role` (string): Message role (user, assistant, system)
  - `parent_id` (string, optional): ID of the parent message
  - `metadata` (object): Message metadata
  - `created_at` (string): ISO 8601 timestamp when the message was created

## Error Codes

| Code | Description |
|------|-------------|
| `404` | Not Found - Conversation does not exist |
| `403` | Forbidden - User does not have access to this conversation |
| `401` | Unauthorized - Invalid or missing API key |
| `500` | Internal Server Error - System error |

## Rate Limiting

Conversation retrieval: Rate limited

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# Get conversation details
result = client.conversations.get("123e4567-e89b-12d3-a456-426614174000")
print(f"Conversation name: {result['name']}")
print(f"Number of messages: {len(result['messages'])}")
```

### JavaScript

```javascript
const { r2rClient } = require("r2r-js");

const client = new r2rClient("your-api-key");

// Get conversation details
const response = await client.conversations.retrieve({
    id: "123e4567-e89b-12d3-a456-426614174000"
});
console.log(`Conversation name: ${response.name}`);
console.log(`Number of messages: ${response.messages.length}`);
```

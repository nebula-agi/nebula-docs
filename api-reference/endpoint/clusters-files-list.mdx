# POST `/clusters/files/list`

Lists all files in the root folder of a cluster.

## Request
- **Body (JSON):**
  - `cluster` (string, required): Cluster name or ID
- **Headers:**
  - `Authorization: Bearer <token>` or `Authorization: ApiKey <key>`

### Example (cURL)
```bash
curl -X POST https://api.nebulacloud.app/clusters/files/list \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"cluster": "my-cluster"}'
```

## Response
```json
{
  "items": [
    {
      "id": "abc123...",
      "name": "document.pdf",
      "isFolder": false,
      "parentFolderId": "ROOT",
      "parentFolderName": "Root",
      "lastModified": "2024-06-01T12:00:00Z",
      "uploadedBy": "Unknown",
      "uploadedByEmail": "",
      "size": "12345",
      "contentType": "application/pdf",
      "status": "completed"
    }
  ],
  "empty": false
}
```

## Error Responses
- `404`: Cluster not found
- `500`: Internal server error

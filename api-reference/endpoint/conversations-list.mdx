# List Conversations

Retrieve a list of conversations for the authenticated user.

<RequestExample>
```bash
curl -X GET "https://api.r2r.ai/v3/conversations?offset=0&limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "results": [
      {
        "id": "123e4567-e89b-12d3-a456-426614174000",
        "name": "My Conversation",
        "user_id": "user-123",
        "created_at": "2024-01-15T10:30:00Z",
        "updated_at": "2024-01-15T10:30:00Z"
      }
    ],
    "total": 1,
    "offset": 0,
    "limit": 10
  }
}
```
</ResponseExample>

**Description:** Returns a paginated list of conversations that belong to the authenticated user.

**Authentication:** Required

**Query Parameters:**
- `ids` (array of strings, optional): A list of conversation IDs to retrieve. If not provided, all conversations will be returned.
- `offset` (integer, optional, default: 0): Number of objects to skip. Must be >= 0.
- `limit` (integer, optional, default: 100): Limit on the number of objects to return. Must be between 1 and 1000.

**Response Fields:**
- `results` (array): Array of conversation objects
  - `id` (string): Unique identifier for the conversation
  - `name` (string): Conversation name
  - `user_id` (string): ID of the user who owns the conversation
  - `created_at` (string): ISO 8601 timestamp when the conversation was created
  - `updated_at` (string): ISO 8601 timestamp when the conversation was last updated
- `total` (integer): Total number of conversations
- `offset` (integer): Current offset
- `limit` (integer): Current limit

## Error Codes

| Code | Description |
|------|-------------|
| `400` | Bad Request - Invalid query parameters |
| `401` | Unauthorized - Invalid or missing API key |
| `500` | Internal Server Error - System error |

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# List conversations
result = client.conversations.list(
    offset=0,
    limit=10
)
print(f"Found {result['total']} conversations")
```

### JavaScript

```javascript
const { r2rClient } = require("r2r-js");

const client = new r2rClient("your-api-key");

// List conversations
const response = await client.conversations.list({
    offset: 0,
    limit: 10
});
console.log(`Found ${response.total} conversations`);
```

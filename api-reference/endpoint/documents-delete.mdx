# Delete Document

Delete a document and all its associated data.

<RequestExample>
```bash
curl -X DELETE "https://api.r2r.ai/v3/documents/b4ac4dd6-5f27-596e-a55b-7cf242ca30aa" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": true
}
```
</ResponseExample>

**Description:** Permanently deletes a document and all its associated chunks, embeddings, and metadata. This action cannot be undone. Only the document owner can delete the document.

**Authentication:** Required

**Path Parameters:**
- `id` (string, required): The unique identifier of the document to delete

**Response Fields:**
- `data` (boolean): True if the document was successfully deleted

## Error Codes

| Code | Description |
|------|-------------|
| `404` | Not Found - Document does not exist |
| `403` | Forbidden - User is not the owner of this document |
| `401` | Unauthorized - Invalid or missing API key |
| `500` | Internal Server Error - System error |

## Rate Limiting

Document deletion: Rate limited

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# Delete document
result = client.documents.delete("b4ac4dd6-5f27-596e-a55b-7cf242ca30aa")
if result:
    print("Document deleted successfully")
```

### JavaScript

```javascript
const { r2rClient } = require("r2r-js");

const client = new r2rClient("your-api-key");

// Delete document
const response = await client.documents.delete({
    id: "b4ac4dd6-5f27-596e-a55b-7cf242ca30aa"
});
if (response) {
    console.log("Document deleted successfully");
}
```

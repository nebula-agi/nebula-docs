# Health Check

Check the health status of the R2R system.

<RequestExample>
```bash
curl -X GET "https://api.r2r.ai/v3/health" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": {
    "message": "ok"
  }
}
```
</ResponseExample>

**Description:** Returns the current health status of the R2R system.

**Authentication:** Not required

**Response Fields:**
- `message` (string): Health status message ("ok")

## Error Codes

| Code | Description |
|------|-------------|
| `500` | Internal Server Error - System error |

## Rate Limiting

Health check: No rate limiting

## SDK Examples

### Python

```python
from r2r import R2RClient

client = R2RClient(api_key="your-api-key")

# Check system health
health = client.system.health()
print(f"System status: {health['message']}")
```

### JavaScript

```javascript
const { r2rClient } = require("r2r-js");

const client = new r2rClient("your-api-key");

// Check system health
const health = await client.system.health();
console.log(`System status: ${health.message}`);
```

# Quickstart with Nebula

Nebula makes it easy to add persistent, context-aware memory to your AI agents and applications. Follow these steps to get up and running:

## 1. Get Your API Key

Sign up and create your project at [Nebula Dashboard](https://dashboard.nebulacloud.app) to obtain your API key.

## 2. Install the SDK

Nebula offers SDKs for multiple languages. For Node.js:

```bash
npm install @nebulacloud/sdk
```

For Python:

```bash
pip install nebula-sdk
```

## 3. Store Your First Memory

```js
// JavaScript/TypeScript Example
import nebula from '@nebulacloud/sdk';

await nebula.store({
  agentId: "agent-123",
  content: "Customer prefers email communication",
  metadata: { userId: "user-456", source: "chat" }
});
```

## 4. Retrieve Knowledge

```js
const memories = await nebula.retrieve({
  agentId: "agent-123",
  query: "communication preferences",
  limit: 5
});
```

## 5. Try Advanced Retrieval (Agentic Layers)

Nebula supports multiple retrieval layers for different use cases:

```js
// Simple semantic retrieval
const memories = await nebula.retrieve({
  agentId: "agent-123",
  query: "customer preferences",
  retrievalType: "simple"
});

// Planning layer
const memories = await nebula.retrieve({
  agentId: "agent-123",
  query: "compile product recommendations based on past purchases",
  retrievalType: "planning"
});

// Reasoning layer
const memories = await nebula.retrieve({
  agentId: "agent-123",
  query: "why did the customer change their mind about product X?",
  retrievalType: "reasoning"
});

// Deep research
const memories = await nebula.retrieve({
  agentId: "agent-123",
  query: "analyze customer sentiment trends over the past year",
  retrievalType: "deepResearch",
  maxDepth: 3
});
```

> See the [API Reference](/api-reference/introduction) for more details and advanced usage.

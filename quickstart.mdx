# Quick Start Guide

<Warning>You'll need a Nebula API key before starting. Get one at [trynebula.ai](https://trynebula.ai)</Warning>

Get up and running with Nebula in minutes.

## Prerequisites

- Python 3.8+ or Node.js 16+
- A Nebula account at [trynebula.ai](https://trynebula.ai)

## 1. Install

```bash
# Python
pip install nebula-client

# Node.js
npm install @nebula-ai/sdk

# Go
go get github.com/nebula-ai/nebula-client-go
```

## 2. Initialize Client

<CodeGroup>

```python client.py
from nebula_client import NebulaClient

# With API key
client = NebulaClient(api_key="your-api-key-here")

# Or set NEBULA_API_KEY environment variable
# client = NebulaClient()
```

```javascript client.js
const { NebulaClient } = require('@nebula-ai/sdk');

const client = new NebulaClient({
  apiKey: 'your-api-key-here'
});
```

 

</CodeGroup>

## 3. Store & Search

<CodeGroup>

```python store_and_search.py
from nebula_client import NebulaClient

client = NebulaClient(api_key="your-api-key-here")

# Create a collection
collection = client.create_collection(name="my_notes", description="Personal notes")

# Store a memory
memory_id = client.store_memory({
    "collection_id": collection.id,
    "content": "Machine learning is transforming healthcare",
    "metadata": {"topic": "AI", "importance": "high"}
})

# Search memories
results = client.search(
    query="machine learning healthcare",
    collection_ids=[collection.id],
    limit=5
)

for result in results:
    print(f"Score: {result.score:.2f}")
    print(f"Content: {result.content}")
```

```javascript store_and_search.js
const { NebulaClient } = require('@nebula-ai/sdk');

const client = new NebulaClient({
  apiKey: 'your-api-key-here'
});

async function storeAndSearch() {
  // Create a collection
  const collection = await client.createCollection({
    name: 'my_notes',
    description: 'Personal notes'
  });

  // Store a memory
  const memoryId = await client.storeMemory({
    collection_id: collection.id,
    content: 'Machine learning is transforming healthcare',
    metadata: { topic: 'AI', importance: 'high' }
  });

  // Search memories
  const results = await client.search({
    query: 'machine learning healthcare',
    collection_ids: [collection.id],
    limit: 5
  });

  results.forEach(result => {
    console.log(`Score: ${result.score?.toFixed?.(2)}`);
    console.log(`Content: ${result.content}`);
  });
}

storeAndSearch();
```

</CodeGroup>

## What You'll Get Back

When you store a memory, Nebula returns the memory ID:

```python
# Returns the memory ID
memory_id = "eng_abc123def456"
```

When you search, you get a list of relevant results with scores:

```python
# Search results structure
[
  {
    "memory_id": "eng_abc123def456",
    "content": "Machine learning is transforming healthcare",
    "score": 0.95,
    "metadata": {"topic": "AI", "importance": "high"},
    "created_at": "2024-01-15T10:30:00Z"
  },
  {
    "memory_id": "eng_xyz789ghi012",
    "content": "AI models improve diagnostic accuracy",
    "score": 0.87,
    "metadata": {"topic": "AI", "importance": "high"},
    "created_at": "2024-01-14T09:15:00Z"
  }
]
```

<Tip>Scores range from 0-1, with higher scores indicating better semantic matches to your query.</Tip>

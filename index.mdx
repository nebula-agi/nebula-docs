# Welcome to Nebula

Nebula is the persistent memory layer for AI agents and applications. It enables your agents to store, retrieve, and share knowledge across every interaction—empowering smarter, more context-aware AI.

## How Nebula Works

<CardGroup cols={2}>
  <Card title="1. Collect Knowledge" icon="brain">
    Your AI agents gather information during user interactions, creating valuable knowledge.
  </Card>
  <Card title="2. Store in Nebula" icon="database">
    This knowledge is securely stored and indexed in Nebula's distributed memory system.
  </Card>
  <Card title="3. Create Connections" icon="network-wired">
    Nebula automatically discovers relationships between pieces of information, connecting the dots.
  </Card>
  <Card title="4. Retrieve & Share" icon="share-nodes">
    Agents can instantly access and share relevant knowledge for future interactions.
  </Card>
</CardGroup>

## Key Features

- **Persistent Memory:** Store and retrieve knowledge across sessions without losing context.
- **Simple API Integration:** Works with any AI model or framework via a REST API.
- **Knowledge Sharing:** Enable agents to access collective knowledge across your system.
- **Agentic Retrieval:** From simple lookups to deep research and reasoning.
- **MCP Support:** Integrate with any LLM using the Model Context Protocol.
- **High Performance:** Sub-100ms retrieval, even at scale.
- **Enterprise Security:** SOC 2 compliant, encrypted, and access-controlled.
- **Retrieval Depth Control:** Choose from simple context to comprehensive research.
- **Usage Analytics:** Monitor memory usage, retrieval quality, and performance.
